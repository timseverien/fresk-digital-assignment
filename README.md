# Tapedeck code challenge

## Installation

Run `npm install`

## Run application

Before running the app, we need an API key for the Tapedeck API.

Create a `.env.local` file in the root directory:

```
TAPEDECK_API_KEY=[put key here]
```

Next, run `npm run dev`

## Running tests

To run the tests, run:

```
npm test
```
